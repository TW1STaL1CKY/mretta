**Realm:** Client

<br>

## Description
Called when drawing the alert triggered using mretta_alerts.Display.<br>
<br><br>

## Arguments
1. **[table](https://wiki.facepunch.com/gmod/table) alertInfo**<br>
&ensp;&ensp;The alert data, including alert text and end timestamp.
<br>

## Returns
1. **[boolean](https://wiki.facepunch.com/gmod/boolean)**<br>
&ensp;&ensp;If `true`, the default Mretta alert HUD will not be drawn.
